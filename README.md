The Lorenz Attractor — a solution to the Lorenz system of ODEs,
originally derived from simplified atmospheric convection equations.

dx/dt = σ(y − x)       σ = 10
dy/dt = x(ρ − z) − y   ρ = 28
dz/dt = xy − βz        β = 8/3

Sensitive dependence on initial conditions.
Two nearly identical starting points diverge completely over time.
This is deterministic chaos — and it's beautiful.
